## A propos de ce site

Ce site web est mon portfolio academique personnel, presentant mes recherches, publications et activites professionnelles. Il est concu pour etre epure, accessible et multilingue.

## Technologies

Ce site est construit avec les technologies suivantes :

- [Jekyll](https://jekyllrb.com/) - Un generateur de sites statiques ecrit en Ruby
- [multi-language-al-folio](https://github.com/george-gca/multi-language-al-folio) - Un fork multilingue du theme al-folio
- [al-folio](https://github.com/alshedivat/al-folio) - Un theme Jekyll oriente academique
- [Bootstrap](https://getbootstrap.com/) - Framework CSS pour le design responsif
- [Font Awesome](https://fontawesome.com/) - Bibliotheque d'icones
- [Academicons](https://jpswalsh.github.io/academicons/) - Jeu d'icones academiques
- [MathJax](https://www.mathjax.org/) - Rendu de notation mathematique

## Hebergement

Le site web est heberge sur [GitHub Pages](https://pages.github.com/) avec un domaine personnalise.
