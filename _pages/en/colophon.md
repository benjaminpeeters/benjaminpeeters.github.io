## About This Website

This website serves as my personal academic portfolio, showcasing my research, publications, and professional activities. It is designed to be clean, accessible, and multilingual.

## Technologies

This site is built with the following technologies:

- [Jekyll](https://jekyllrb.com/) - A static site generator written in Ruby
- [multi-language-al-folio](https://github.com/george-gca/multi-language-al-folio) - A multilingual fork of the al-folio theme
- [al-folio](https://github.com/alshedivat/al-folio) - An academically-oriented Jekyll theme
- [Bootstrap](https://getbootstrap.com/) - CSS framework for responsive design
- [Font Awesome](https://fontawesome.com/) - Icon library
- [Academicons](https://jpswalsh.github.io/academicons/) - Academic icon set
- [MathJax](https://www.mathjax.org/) - Mathematical notation rendering

## Hosting

The website is hosted on [GitHub Pages](https://pages.github.com/) with a custom domain.
